# Xlab进阶项目

项目为使用React+SWR+Trpc制作一个简单的聊天室

## 配置设定

在`./src/server/db.ts`中填入数据库的信息

随后使用`yarn install`安装依赖

## 项目启动

使用`yarn run dev`启动网页程序

使用`yarn run start`启动后端程序
